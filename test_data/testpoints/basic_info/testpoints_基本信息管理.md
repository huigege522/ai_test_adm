**「自动化是否覆盖」取值说明**（对照现有 pytest）：


| 取值     | 含义                                                          |
| ------ | ----------------------------------------------------------- |
| 已覆盖    | 存在可执行用例且含 HTTP/业务/列表回显等有效断言                                 |
| 部分覆盖   | 仅有接口契约或弱断言、多 TC 合并、仅 E2E 可进页、或 UI 用例由 API 契约替代（如 TC212/213） |
| skip占位 | `test_basic_info_blocked_skips.py` 或 `pytest.skip` 占位       |
| 未覆盖    | 尚无对应用例                                                      |
